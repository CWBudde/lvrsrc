# lvrsrc

`lvrsrc` is a pure-Go RSRC/VI toolkit focused on safe parsing, structural validation,
and preserving rewrites for LabVIEW files.

## Current status

- Phase 0 scaffolding and documentation are in place.
- Phase 1 implementation has started with the low-level binary reader package.
- Sample corpus ingestion/oracle baselines are pending user-provided inputs.
